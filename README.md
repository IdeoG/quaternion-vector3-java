Quaternion and Vector3 library implemented in Java
===

Getting started
---

Repository contains similar to Unity3D library operations and much more.

Quick overview
---

**Quaternion methods:**

- euler to quaternion
- quaternion to euler
- normalize
- conjugate
- cross product
- rotate vector
- calculate twist
- to string

**Vector3 methods**

- normalize
- add
- minus
- multiply by float
- cross product
- dot product
- calculate vector rotation quaternion
- projection
- norm
- to string
